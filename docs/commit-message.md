# Commit message conventions

See [docs/commit-message.md](../../../docs/commit-message.md) for shared rules (format, body, anti-patterns).

## Scopes for this package

Use the **source filename without extension** as scope:

`array`, `error`, `html`, `set`, `string`, `datetime`, `unicode`, `markdown`, `types`, `lock-fn`, `flat`, `iterator`, `map`, `misc`, `regex`, `zod`, `algo`

Also: `testing` for test infrastructure.
